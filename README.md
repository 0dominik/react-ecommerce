# Olios shop

## Overview

This is a ecommerce React app. It allows you to browse or search products and add them to cart. Template is from symu.co

## Technologies

- React
- Styled components
- Firebase (authentication)
- Firestore
- Apollo/GraphQL
- React Router DOM
